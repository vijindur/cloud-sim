# Trace Source References

The repository includes expanded local sample traces for development and reproducibility.
For research-grade studies, use official traces from:

1. **Google Cluster Data (2019 release)**
   - https://github.com/google/cluster-data

2. **Alibaba Cluster Trace Program**
   - https://github.com/alibaba/clusterdata

These references should be used when replacing local sample data with full, real-world traces.
